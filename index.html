About This Lens

This custom lens maps the MAS Project MindForge AI Risk Management and Governance Framework (January 2026) into the AWS Well-Architected Tool. It enables Financial Services Institutions to systematically assess their AI governance posture using AWS-native review workflows.

1. AI Oversight

Scope, responsibilities, and the three lines of defence model.

2. AI Risk Management

Policies, enterprise risk, third-party risk, use case risk, and inventory.

3. AI Lifecycle Management

Context, data, build, deploy, and monitor/change.

4. Enablers

Skills, knowledge, culture, and infrastructure.

Download Lens JSON View on GitHub

How to Import

  1. Download the lens JSON file using the button above.
  2. Open the AWS Well-Architected Tool in your AWS account.
  3. Navigate to Custom Lenses and click Create custom lens.
  4. Upload the mindforge-lens.json file.
  5. Review the lens preview and publish it.
  6. Apply the lens to a workload and start your AI governance review.
